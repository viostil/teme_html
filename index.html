Open the MS Word document that contains your resume. Click the Office button (the colorful logo in a circle found in the upper left corner of your MS Word window). Click Save As and select Other Formats. At the bottom of the window that pops up on your screen, type in a new name for this document in the File Name field, such as "ResPlainText." Under this is the Save As Type pull-down menu. Scroll down this list to select "Plain Text (*.txt)." Click Save to perform the conversion, while still preserving the original document (.docx) you use for your hardcopy resume. When the File Conversion window appears, click OK without changing any of the settings.
